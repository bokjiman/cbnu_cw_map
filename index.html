➕ 내 자원 카드 만들기

내가 나눠줄 수 있는 것과 받고 싶은 것을 한 줄로 적어 우리과 자원망에 올려요. 우리는 서로의 자원이니까요 💛

이름 (또는 별명)
나를 닮은 이모지
GIVE 내가 나눠줄 수 있어요
WANT 내가 받고 싶어요
카테고리는 여러 개 골라도 돼요. (나눌 것·받을 것 중 하나 이상)

🗂 전체 참여자 Raw 데이터

현재 명단의 원본 데이터(JSON)예요. 아래 칸을 누르면 직접 편집할 수 있고, 적용을 누르면 자원망에 바로 반영됩니다. 읽기 전용 — 누르면 편집

📜 버전 · 수정 로그

이 파일의 버전과, 추가·삭제·불러오기 등 모든 변경 이력이에요. 가장 위가 최신 변경입니다.

❓ 사용법 — 우리과 자원망

  • ＋ 내 자원 카드 만들기 (오른쪽 아래 파란 버튼 또는 상단 노란 버튼) → 이름·이모지·나눌 것·받고 싶은 것을 적어 올립니다.
  • 친구 노드 누르기 → 내가 중심으로 이동하고, 왼쪽=나에게 줄 친구(받음), 오른쪽=내가 줄 친구(줌)가 화살표로 흐릅니다.
  • 자동 저장 → 추가·삭제한 내용은 이 기기 브라우저에 자동 저장돼 다시 열어도 남습니다.
  • 🗂 Raw → 참여자 전체 원본 데이터를 보고, 칸을 눌러 직접 편집한 뒤 ‘적용’할 수 있어요(담당자용).
  • 📤 파일로 저장 (Export) → 현재 상태 전체를 .json 파일로 내려받습니다. (수정 로그도 함께 저장 → 최종본 식별 가능)
  • 📥 불러오기 (Import) → 받은 .json을 합칩니다. 같은 이름은 최신으로 덮어쓰며, 새 이름은 추가됩니다.
  • 🔗 링크 → 현재 명단을 담은 공유 링크를 복사합니다(호스팅 시 가장 매끄러움).
  • 📜 로그 → 버전과 변경 이력을 확인합니다.

배포·취합 권장 절차: ① 학생들이 각자 카드를 추가 → ② 📤 파일로 저장한 .json을 담당자에게 전달 → ③ 담당자가 📥 불러오기로 모두 합친 뒤 → ④ 합쳐진 파일/링크를 다시 배포. 매번 ‘📜 로그’의 최신 시각으로 최종본을 확인하세요.
